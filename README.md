PROJECT SUMMARY:  
This is Recylcying Resource Hub, a tool used to help you find more about recycling and you're impact with it!

FEATURES:  
A persistent menu system with five subjects about recycling.  
Returned information about recycling for the user  
Tracking of the user's impact through amount cycled and energy saved.    
  
STRUCTURE:  
One main class that runs a persistent while loop until exited, with the menu going a layer deeper into each specific resource when it is called too due to the user's input.  
Within the five resources, it'll run it's specific task and then wait to allow the user time to read before returning to the menu cycle.  
Upon the user's request, the program will terminate by the user inputting the associated input.  
  
HOW TO USE:  
Open the program inside of a local application that runs code, and run the program!
